# 📱 Invoice Android App

A simple, fast, and user-friendly Android application for managing invoices and customer billing.

---

## ✨ Features

- 🧾 Create and manage invoices easily  
- 👤 Add and store customer details  
- 💾 Save invoice history locally  
- 🔍 Search and view past invoices  
- 📊 Simple and clean UI design  

---

## 🛠️ Tech Stack

- Java / Kotlin (Android Development)  
- XML (UI Design)  
- SQLite / Room Database  
- Android Studio
